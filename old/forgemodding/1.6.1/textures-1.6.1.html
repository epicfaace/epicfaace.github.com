Textures - 1.6.1

Block/Item Icons

Add this to the item's class:

public void registerIcons(IconRegister iconRegister)
{
	this.itemIcon = iconRegister.registerIcon("GenericMod:"+this.getUnlocalizedName());
}

To the block's class:

    @Override
    public void registerIcons(IconRegister iconRegister) {
        this.blockIcon = iconRegister.registerIcon("GenericMod:"+this.getUnlocalizedName());
    }

Basically, the mods folder has been renamed to assets. The file path is now mcp/src/minecraft/assets/epic.zirc/textures/blocks|items/[unlocalizedname].
At the beginning, add item. to the name or tile. for a block.

Also, the unlocalized name of an item is also prefixed by item., and maybe tile. for blocks?
