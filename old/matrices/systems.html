Cramer's Rule

Cramer's rule is an efficient way of solving systems of 2 and 3+ variables.todo: add color to diagrams

2 variables

$$ax+by=e$$ $$cx+dy=f$$ $$A=\left[ \begin{array}{cc}a&b\\c&d\end{array} \right]$$ $$x=\frac{\left[ \begin{array}{cc}e&b\\f&d\end{array} \right]}{|A|}, y=\frac{\left[ \begin{array}{cc}a&e\\c&f\end{array} \right]}{|A|}$$ To solve for a variable, replace the column containing that variable's coefficients with the coefficients which are just numbers.

3+ variables

Very similar to 2 variables: $$ax+by+cz=j$$ $$dx+ey+fz=k$$ $$gx+hy+iz=l$$ $$A=\left[ \begin{array}{ccc}a&b&c\\d&e&f\\g&h&i\end{array} \right]$$ $$x=\frac{\left[ \begin{array}{ccc}j&k&l\\d&e&f\\g&h&i\end{array} \right]}{|A|}, y=\frac{\left[ \begin{array}{ccc}a&b&c\\j&k&l\\g&h&i\end{array} \right]}{|A|}, z=\frac{\left[ \begin{array}{ccc}a&b&c\\d&e&f\\j&k&l\end{array} \right]}{|A|}$$ Similarly, this can apply to systems of equations for 4+ variables, although most people use computers/calculators for those.

Using Inverses

$$ax+by=e$$ $$cx+dy=f$$ Create a matrix equation and solve: $$\left [ \begin{array}{cc}a&b\\c&d\\ \end{array} \right]\left [ \begin{array}{c}x\\y\end{array} \right]=\left [ \begin{array}{c}e\\f\end{array} \right]$$ $$\left [ \begin{array}{c}x\\y\end{array} \right]=\left[ \begin{array}{cc}a&b\\c&d\end{array} \right]^{-1}\left [ \begin{array}{c}e\\f\end{array} \right]$$
